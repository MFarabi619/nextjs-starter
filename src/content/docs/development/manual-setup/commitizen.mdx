## Installation

<InstallTabs items={['npm', 'pnpm', 'yarn', 'bun']}>

```bash tab="npm" title="Terminal"
npm i -D commitizen @commitlint/cz-commitlin inquirer
```

```bash tab="pnpm" title="Terminal"
pnpm i -D commitizen @commitlint/cz-commitlint inquirer
```

```bash tab="yarn" title="Terminal"
yarn i -D commitizen @commitlint/cz-commitlint inquirer
```

```bash tab="bun" title="Terminal"
bun i -D commitizen @commitlint/cz-commitlint inquirer
```

</InstallTabs>

## Configuration


```bash title="Terminal"
touch .cz.json # In root directory
echo '{ "path": "@commitlint/cz-commitlint" }' > .cz.json
```

### Add `package.json` script

```diff title="package.json"
...
  "scripts": {
    "build": "next build",
    "dev": "next dev",
    "start": "next start",
    "lint": "eslint .",
    "lint:fix": "eslint . --fix",
    "lint:inspect": "npx @eslint/config-inspector",
    "prepare": "husky"
+   "commit": "cz" // [!code highlight]
  },
  ...
```
