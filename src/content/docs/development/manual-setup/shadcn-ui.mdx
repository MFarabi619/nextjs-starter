## Installation

<Callout type="error">
While the Fumadocs UI design system is inspired by [Shadcn/ui](https://ui.shadcn.com/) and also uses Tailwind CSS, it does not use the same tokens and is therefore not compatible. Installing Shadcn/ui will overwrite the current Tailwind CSS configuration, which will break Fumadocs UI.

Next.js allows for separate Root layouts for different routes, which will allow us to use both design systems without conflicts.

  </Callout>

### Separate design systems

Create `(home)/` under `app/`. **Copy** `global.css`, `layout.tsx`, and `page.tsx` from `app/` into `app/(home)/`.

Then **cut/move** `layout.config.tsx`, `source.ts`, and `global.css` into `app/docs/`.

Delete `layout.tsx` in `app/`.

### Run Shadcn/ui install script

Follow the below steps and cross-check with the instructions in the [Shadcn/ui docs](https://ui.shadcn.com/docs/installation/next) to install Shadcn.
Step 1 on their docs can be skipped, as we've already set up a Next.js project.

```bash title="Terminal"
pnpm dlx shadcn-ui@latest init
```

