## Installation

<InstallTabs items={['npm', 'pnpm', 'yarn', 'bun']}>

```bash tab="npm" title="Terminal"
npm i -D semantic-release
```

```bash tab="pnpm" title="Terminal"
pnpm i -D semantic-release
```

```bash tab="yarn" title="Terminal"
yarn i -D semantic-release
```

```bash tab="bun" title="Terminal"
bun i -D semantic-release
```

</InstallTabs>

## Configuration

```bash title="Terminal"
touch release.config.ts
echo "import type { GlobalConfig } from 'semantic-release'\n\nconst config: Partial<GlobalConfig> = {\n  branches: ['main']\n}\n\nexport default config" > release.config.ts
```
